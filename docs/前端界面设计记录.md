# 前端界面设计记录

## 2026-06-14：预测工作台信息架构调整

### 本次目标

把首页从通用世界杯预测门户，调整为面向赌球用户和专业博主/分析号的每日预测工作台。

### 已完成内容

- 首屏改为“今日预测工作台”，突出今日重点比赛。
- 首页新增最可能比分 Top 3、比分矩阵、大小球 2.5、BTTS 和模型公平概率。
- 新增市场价格源状态提示，当前明确标注为待接入，不伪装真实市场价格。
- 新增今日可讲选题，服务专业博主/分析号的内容生产。
- 路径传导模块前置，展示单场赛果如何影响小组和冠军概率。
- 单场快速预览接入模型公平概率、进球市场和比分矩阵。
- 单场详情页的完整预测包调整为：比分矩阵、进球市场、模型公平概率、路径传导和博主素材。

### 关键决策

- 第一优先级不再是泛用户的冠军榜，而是单场比分分布、90 分钟胜平负、进球市场视角和路径影响。
- 赌球用户需要的是模型概率和市场价格的差值，但在真实授权价格源接入前，只展示模型公平概率和“市场价格源待接入”。
- 专业博主/分析号需要可讲述素材，所以首页保留选题卡和变化原因，而不是只给数字。
- 页面继续使用“概率分析”定位，不出现投注建议。

### 未解决问题

- 还没有接入授权市场价格源，因此无法展示真实模型-市场差值。
- 还没有球员进球、牌数、角球等 props 预测。
- 单场页完整预测仍依赖付费解锁状态，未做内容截图导出。

### 下一步

- 接入市场价格源后，补“模型概率 vs 市场价格”差值模块。
- 增加博主可截图预测卡和短文案导出。
- 补球员 props 前，先确认数据授权和合规边界。
